Incident Reflection App

The Incident Reflection App is a web-based tool that supports personal reflection and improvement in community service and first response work. Its main function is to record events as they occur during a shift, so that you can review and reflect on them in detail afterwards.

The main features of the app are as follows:

  • Incident Logging: You can record key details, including date, time, severity, case type, interventions, and clinical roles, for each incident.
  • Reflective Practice: The app includes a self-reflection tool that uses tags for common areas of improvement, including communication, scene control, and teamwork.
  • Emotional Awareness: You can document your emotional state to help identify patterns in your performance and well-being.

Disclaimer: This application was developed as a personal learning project and for individual reflection. It is a lightweight tool created primarily for experimentation and practice, and is not intended to replace official incident reporting systems, clinical documentation, or organisational record-keeping. Patient or personally identifiable information should not be recorded in this application.

Vehicle Pre-Check Application

The Vehicle Pre-Check Application is a basic tool that helps carry out vehicle pre-checks.

Key features include:

  • Comprehensive Inspections: It provides categorised checklists for different parts of the vehicle, including the engine bay, exterior, tires, and interior controls.
  • Live Status Tracking: As you move through the inspection, a real-time summary counter helps you keep track of which items are in good condition and which need attention.
  • Offline Capability (PWA): Because the app is built as a Progressive Web App, you can complete inspections even in areas with poor cellular reception
  • Digital Logging: The app replaces paper forms by storing inspection data locally and allowing you to export reports as PDFs for audits and maintenance records.

Disclaimer: This project was built as a personal learning exercise. It is a lightweight tool intended for experimentation and personal use, and no guarantees are made regarding accuracy, completeness, or suitability for professional use.
