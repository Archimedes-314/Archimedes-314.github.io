The Axioms

  1. \(0\in\N_0\)
  2. \(\forall n\in\N_0, \ \S(n)\in\N_0\)
  3. \(\forall n\in\N_0, \ \S(n)\neq0\)
  4. \( \forall n,m\in\N_0, \ \S(n)=\S(m)\implies n=m \)
  5. \(\forall A\subseteq\N_0, \ (0\in A \ \land \ (\forall n \in A, \ \S(n)\in A) ) \implies A=\N_0\)

In this section, we will look into the meaning and purpose of each axiom.

Notation

There are two common definitions of the set \(\N\). Those are as follows:

  • \(\N := \{1, 2, 3, 4, \ldots\}\)
  • \(\N := \{0, 1, 2, 3, 4, \ldots\}\)

The definition you use depends on context. But within analysis branches, it's common to define \(\N := \{1, 2, 3, 4, \ldots\}\), and \(\N_0 := \{0, 1, 2, 3, 4, \ldots\}\). That is the definition we will be using.

We will also be using the function \(\S(n)\) while exploring the Peano Axioms and the construction of the reals. This function is called the successor function, and is defined as follows: \begin{align*} \S : \N_0 &\to \N_0\\ n &\mapsto n+1 \end{align*} So the successor of any natural number returns that number plus \(1\).

The Axiom of Induction

What it's saying

Now we'll look into the fifth axiom: \(\underbrace{\forall A\subseteq\N_0}_{\text{Arbitrary Subset}}, \ \underbrace{(0\in A \ \land \ (\forall n \in A, \ \S(n)\in A) )}_{\text{Conditions}} \implies \underbrace{A=\N_0}_{\text{Conclusion}}\)

This axiom has three main components:

  1. Choose any subset \(A \subseteq \N_0\).
  2. Assume it satisfies the two stated conditions.
  3. Conclude the necessary result if the conditions are met.

Taken together, these say that if an arbitrary subset of the naturals satisfies the two conditions, then that subset must be equal to \(\N_0\).

The first condition is \(0\in A\), requiring our subset to contain \(0\). The next condition is \(\forall n\in A, \ \S(n)\in A\), meaning that for any member of \(A\), it's successor is also in \(A\).

So if \(0\) is in \(A\) and the successor of all numbers in \(A\) is also in \(A\), that requires \(A=\{0, \S(0), \S(\S(0)), \ldots\}\), which is exactly \(\N_0\).

Why it's important

Say we have some proposition \(P\), and we want to prove that the proposition is true for all natural numbers. A simple way to do this is to start with the set \(A:=\{n\in\N_0 \ : \ P(n)\text{ is true}\}\) which contains the set of naturals numbers for which \(P\) is true.

First, we can show \(P(0)\) is true. That is, the proposition is true when the argument is \(0\). If that holds, we can then say \(0\in A\).

Next, we assume that \(n \in A\). That is, we assume \(P(n)\) is true for some arbitrary natural number \(n\). Under this assumption, we then show that \(P(\S(n))\) is also true. If we can prove this in a way that doesn't restrict \(n\) to any specific value, that we can claim \(n\in A \implies \S(n)\in A\)

Once we've shown that \(0 \in A\) and \(\forall n \in A,\ \S(n) \in A\), then the axiom of induction tells us that \(A = \mathbb{N}_0\). Therefore, the proposition must be true for all natural numbers.
