Overview

In everyday conversations, the meaning of what we say can change depending on context and emphasis. For example, take the two following questions:

  • Why are you here?
  • Why are YOU here?

In the first question, I am asking why you chose to visit this website rather than looking elsewhere; However, in the second question, I am asking why you, in particular, are here, suggesting that others may be welcome, but you are not.

Although both questions use the same four words, their meanings are completely different; the first question is asked with curiosity, and the second is Confrontational. In mathematics, we want to avoid this kind of ambiguity and to express ideas clearly. To do this, we use mathematical logic, which provides a precise language in which each symbol has a single, well-defined meaning and every statement follows strict rules, ensuring that ideas are expressed clearly and prevent misinterpretation.

Logic forms the foundation of pure mathematics and is a great starting point for learning how mathematics is rigorously built. To get a basic idea of how logic works, we will look at the following example: When you receive a phone call, there are two options: the ringtone plays, and the ringtone doesn't play. This should be determined by whether the phone is set to silent. This situation can be described using logical statements by setting the following variables:

  • \(s\) represents 'Silent mode is on'.
  • \(p\) represents 'The ringtone plays'.
  • \(d\) represents 'The ringtone doesn't play'.

To represent the two options, we can say:

  • If \(s\) is true, then \(d\) is true.
  • If \(s\) is not true, then \(p\) is true

Similarly, in an app, the login button should only function when both the username and password have been entered. If either the username or the password is incorrect, the user should receive an error message. Just like before, we can create logical statements to show the outcome using the following variables:

  • \(u_c\) represents 'The correct username is provided'.
  • \(p_c\) represents 'The correct password is provided'.
  • \(l\) represents 'The user was logged in'.
  • \(e\) represents 'The user received an error'.

We can then say:

  • If \(u_c\) and \(p_c\) are true, then \(l\) is true.
  • If \(u_c\) or \(p_c\) are false, then \(e\) is true.

This type of reasoning, which uses words like 'if', 'then', 'not', 'and', and 'or', forms the basis of logic. With these logical tools, we can precisely determine whether mathematical statements are true or false.

In the following sections, I will introduce the syntax and semantics of propositional logic, including the definition of a proposition, the correct expression of propositions, and the meanings of basic logical symbols. I will then discuss several methods for proving propositions.
