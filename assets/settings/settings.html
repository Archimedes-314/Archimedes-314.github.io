You can toggle dark mode on or off using the button below.

Reduced Motion

Disables animations and transitions across the site. Useful if you are sensitive to motion or prefer a simpler interface. If you want this setting enabled system-wide, consider enabling the "Reduce Motion" option in your system settings.

Show Hidden Working by Default

This setting controls whether the hidden working is shown by default when viewing a page. If enabled, the hidden working will be visible immediately.
